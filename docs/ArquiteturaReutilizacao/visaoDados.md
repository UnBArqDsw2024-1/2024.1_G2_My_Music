## Introdução

__Visão de Dados__

o Documento de Arquitetura de Software Visão de Dados é um documento opcional que abrange a visão de dados da aplicação. No caso da utilização de um Banco de Dados Relacional a visão de dados pode ser um dcoumento de DER ou mER. No caso da utilização de um banco de dados não relacional como o nosso caso, a diagramação pode ser um artefato generico.

## Metodologia

Para o desenvolvimento do documento de visão de dados, primeiramente foi feito um estudo inicial de como deve ser esse documento, para o estudo foi utilizado video aulas e slides da professora Milene Serrano. Foi utilizado como base o diagrama de classe e foi utilizado o site Lucichart.
